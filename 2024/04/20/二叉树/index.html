为什么需要二叉树

因为要模拟递归。

编程是在干嘛？

编程是在管理数据，问题在于怎么个管理法。管理着管理着就发现原来管理本身就是数据。
写代码的过程就是一个递归的过程:

function coding(){
   if (NoError){
       done
   } else {
       coding(change_some_codes)
   }
 }

所以我喜欢git这种东西，这是在模拟人在编码过程中不断后退重来的困境。
